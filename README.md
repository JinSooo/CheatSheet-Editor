# 🤝 手动编译

## 环境要求

- Node.js >= 18.0.0
- pnpm >= 8.5.0

## 开始编译

1. Clone 仓库

   ```bash
   git clone https://github.com/JinSooo/CheatSheet-Editor.git
   ```

2. 安装依赖

   ```bash
   cd CheatSheet-Editor
   pnpm install
   ```

3. 开发调试

   ```bash
   pnpm dev # Run the app in development mode
   ```

4. 打包构建

   ```bash
   pnpm build # Build into installation package
   ```
